Intro to Tables

Tables are good for structuring data in an organized way.

Creating a Table

Table rows and columns are put in between the table tags.

Table Data

Table data uses td tags. Table data are the cells in a table. They are placed inside the table rows.

Table Headings

Table headings use the th tag. They are place in a row by themselves.

Table Borders

Tables can have borders by adding inline styling or in the styles sheet. Do not use the borderattribute as it is deprecated.

Spanning Columns

To allow for a td element to span multiple columns use the colspan attribute.

Spanning Rows

To allow for a td element to span multiple rows use the rowspan attribute.

Table rows use tr tags.

Table Body

The table body or tbody element encompasses the table rows and table data elements.

Table Head

The table head or thead element is the top most row in the table element.

Table Footer

The tfoot element is the bottom of the table.

Styling with CSS

Tables, by default, are very bland. They have no borders, the font color is black, and the typeface is the same type used for other HTML elements.

Just like an HTML document, we can add styles via the tag, class or id.
