HTML Intro

HTML is a text document. It gives structure to a webpage.

How People Access the Web

People use web browsers to access the web through a number of devices. The majority of those are on mobile phones or tablets. The lower percentage of access comes from desktop browsers on a laptop or pc.

When a user access a web page it makes a call to a web server. The webapge is hosted by web hosting company

Screen readers are programs that read out the contentsof a computer screen to a user. This is a feature available to users with disabilities.

How Websites are Created

All websites use HTML and CSS.

In addition to the HTML and CSS some websites uses images and Javascript to make the site more engaging and dynamic.

Larger sites may use a Content Management System or CMS. Even more complex sites such as e-commerce sites will use databases withy complext tables to make requests to and display the information on screen.

  1. Connect to the web via Internet Service Provider
  2. Computer contacts a Domain Name System (DNS) using your Internet Prodvider address
  3. The DNS returns a unique number that will allow your browser to contact the web server of the website you requested.
  4. The web server sends the page back to your browser.

HTML

Elements

Elements are the building blocks of an HTML document.

Element Breakdown

An element is made up an opening tag and a closing tag.

An opening tag has a less-than sign followed by a character (depending on the element needed). After the character comes a greater-than sign. A closing tag starts with a less-than sign followed by a forward slash. Then the same character used in the first tag followed by a greater-than sign.

Element Attributes

Different elements have attributes that add functionality to an element. The structure of the attribute is denoted by the attribute name followed by an = sign. Then, the value of the attribute is placed between quotation marks after the equal sign.

Structure

As stated before, elements make up the building blocks of an HTML document. The head, title and body elements are key to any HTML document.

Head, Title, Body

The head goes before the body and contains the title element. It includes metadata - descriptive elements.

The title comes inside the head element. It is what shows up on the browser tab.

The body element contains all of the markup that is displayed in the browser.

Headings

Headings are elements that denote a title of a section or a div. There are six levels of headings. The largest and most important is h1 all the way down in size to an h6.

Paragraphs

Paragraph elements are one or more sentences that form a sel-contained unit of discourse. Each paragraph is on a new line.
